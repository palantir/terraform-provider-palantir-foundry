# Terraform Provider for Palantir Foundry

The Terraform Provider for Foundry is a Terraform plugin, built with the Terraform plugin framework, used for managing resources within Palantir Foundry. 

For a full list of supported resources and data sources, please refer to the [documentation](https://registry.terraform.io/providers/palantir/palantir-foundry/latest/docs).
